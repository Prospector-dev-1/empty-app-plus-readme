# Investor App

Deal flow platform for discovering and backing startups.

## Features

- Startup discovery (Tinder-style swipe)
- Due diligence tools
- Portfolio tracking
- Market intelligence
- NDA management

## Development

```bash
npm run dev
```

Runs on `http://localhost:5180`

## Building

```bash
npm run build
```

Part of the Fishtank monorepo.
